JavaScript is a high-level programming language that follows the ECMAScript standard.
It was originally designed as a scripting language for websites but became widely
adopted as a general-purpose programming language, and is currently the most popular
programming language in use.

html is a markup language used for structuring and presenting hypertext documents on the World Wide Web.
It was the fifth and final[4] major HTML version that is now a retired World Wide Web Consortium (W3C) recommendation

css is a style sheet language used for specifying the presentation and styling of a document written in
a markup language such as HTML or XML
